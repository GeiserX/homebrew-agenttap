# AGENTS.md — homebrew-agenttap

Homebrew tap for [AgentTap](https://github.com/GeiserX/AgentTap), a macOS menu bar app for network-level AI agent observability.

## Repository Purpose

This repository contains the Homebrew cask formula for installing AgentTap via `brew install --cask agenttap`.

## Cask Formula

The cask at `Casks/agenttap.rb` is automatically updated by the [AgentTap release workflow](https://github.com/GeiserX/AgentTap/blob/main/.github/workflows/release.yml) whenever a new version is tagged.

## Installation

```bash
brew tap geiserx/agenttap
brew install --cask agenttap
```

## Upgrade

```bash
brew upgrade --cask agenttap
```

## Uninstall

```bash
brew uninstall --cask agenttap
```

This removes the app, the privileged helper daemon, and its LaunchDaemon plist.
