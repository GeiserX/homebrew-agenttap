# homebrew-agenttap

Homebrew tap for [AgentTap](https://github.com/GeiserX/AgentTap) — network-level AI agent observability for macOS.

## Install

```bash
brew tap geiserx/agenttap
brew install --cask agenttap
```

## Upgrade

```bash
brew upgrade --cask agenttap
```

## Uninstall

```bash
brew uninstall --cask agenttap
```
