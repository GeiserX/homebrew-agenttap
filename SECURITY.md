## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do NOT open a public issue**
2. Use [GitHub's private vulnerability reporting](https://github.com/GeiserX/homebrew-agenttap/security/advisories/new)
3. Or email: 9169332+GeiserX@users.noreply.github.com

You can expect an initial response within 48 hours. We will work with you to understand and address the issue before any public disclosure.
